## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack and Dependencies
### Server Side
- Environment: `Node.js`
- Database/Cache: `Redis`
### Web Client
- Framework: `React + TypeScript + Vite`
- Styling：`TailwindCSS`
- State Management：`Zustand`
- Data Fetching: `SWR`
